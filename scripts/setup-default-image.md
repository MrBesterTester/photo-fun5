# Setting Up the Default Image

To add your default portrait image:

1. **Save your image file** to: `public/images/default-portrait.jpg`
   - The file must be named exactly: `default-portrait.jpg`
   - Supported formats: JPG, PNG, or WebP

2. **Alternative method** (if you have the image in your clipboard or browser):
   - You can also drag and drop the image file directly into the `public/images/` folder in your file explorer
   - Then rename it to `default-portrait.jpg`

3. **After adding the image**, refresh your browser and the default image should appear automatically when the app loads.

The app will:
- Load the default image on startup
- Allow users to upload their own images via "Replace Image" button
- Allow users to download processed images via the "Save" button
